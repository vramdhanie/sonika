### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/sonika.git
   cd sonika
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```
